When AI Gives Bad Output

🔧
Quick Fixes
  • Too generic? Add more context about YOUR situation. Name, role, specific project.
  • Too long? Add "Keep it under 100 words" or "3 bullet points max."
  • Wrong tone? Paste an example of writing you like and say "Match this style."
  • Hallucinating facts? Ask "Are you confident about these facts? Which ones should I verify?"
  • Ignoring instructions? Start a new conversation. Memory fills up and context gets noisy.
