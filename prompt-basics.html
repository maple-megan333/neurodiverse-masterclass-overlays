Prompt Basics That Actually Work

The 4-element prompt framework for neurodivergent brains.

⏱ Time & Energy
15 min · Any energy · You'll need: Claude.ai + this page
Works with any AI tool. These steps say "Claude" but work in Gemini, ChatGPT, or any AI assistant.

Step 1: The One Pattern That Matters

🔬 Why this works
Structured prompts consistently outperform unstructured requests (White et al., 2023). For ND brains, the framework reduces working memory load.
☁️ Low energy version
Build ONE prompt. Pick whichever category you'd use soonest. Fill in the template. Run it. Save it. That counts.
📘 The 4-Element Prompt Framework
ElementWhat it doesExample
RoleTells AI who to be"You are a direct, warm communicator"
ContextGives background"I need to decline a meeting invite from my boss"
TaskWhat you need"Write a response that's firm but respectful"
FormatHow to deliver"2-3 sentences, casual professional tone"
You don't need all four every time. Adding even one dramatically improves results.

Step 2: Build Your First Prompt — Work

✏️ Do this now
Think of a work task you do regularly. Copy this template into Claude:
Work PromptYou are a [ROLE — e.g., "concise professional writer"].

Context: [YOUR SITUATION — e.g., "I'm a freelance designer sending weekly updates"]

Task: [WHAT YOU NEED — e.g., "Write a project update email. Wireframes done, visual design next."]

Format: [HOW YOU WANT IT — e.g., "3 short paragraphs, warm but professional, under 150 words"]
If output is close but not right: "Make it shorter" · "Make it more casual" · "Make it sound more like me"
⭐ What good looks like
The output feels 70-80% right on first try. Way off? Add more context. Wrong tone? Add a format instruction.

Step 3: Build Two More — Life + Creative

🏠 Life prompt
Life PromptYou are a [ROLE — e.g., "calm, practical life organizer"].

Context: [YOUR SITUATION — e.g., "I have ADHD and struggle with meal planning."]

Task: [WHAT YOU NEED — e.g., "Create a 5-day dinner plan using ingredients I probably have."]

Format: [HOW YOU WANT IT — e.g., "List, one per day, include cook time"]
🎨 Creative prompt
Creative PromptYou are a [ROLE — e.g., "creative brainstorm partner"].

Context: [YOUR SITUATION — e.g., "Newsletter about ND productivity"]

Task: [WHAT YOU NEED — e.g., "Give me 10 topic ideas. Mix practical tips with stories."]

Format: [HOW YOU WANT IT — e.g., "Numbered list, one sentence each, star top 3"]
