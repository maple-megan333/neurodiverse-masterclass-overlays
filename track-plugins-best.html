📁 Filesystem & Local 2 servers
@modelcontextprotocol/server-filesystem ✅ Official
The most-installed MCP server of 2026. Gives Claude path-restricted read and write access to specific directories — safer than running everything through the Bash tool.
npx -y @modelcontextprotocol/server-filesystem ~/workspace
Note: Args restrict access to the listed paths. Never pass the system root (/) or your home directory blindly.

Terminal access plus process management. Useful for system-level automation, but it can do anything your shell can.

Note: Run only on machines where you accept the same risk you'd accept giving an SSH session to a teammate. Not a starter install.

Auth patterns

Three auth methods cover ~99% of the MCP servers in production. Knowing which one a server uses tells you what kind of secret to budget for.

Env var
Used by ~79% of current servers. Stdio servers and most local servers. Set a secret in your shell, reference it as ${env:VAR} in .mcp.json. Lowest setup cost; rotate manually when needed.
OAuth 2.1
Mandated by the 2026 MCP spec for remote servers. The plugin install flow handles the OAuth dance for you — you click "Allow," tokens land in the plugin's secure store, refresh happens silently. Best UX, but only available for HTTP / remote MCP.
Bearer token
Used by services that manage their own tokens. PixelLab, some custom internal MCPs. Token goes in the Authorization header. Treat like an API key — never commit, rotate on any leak.

Top 15 Plugins for ND Work + Life

Plugins are easier to install than individual MCP servers and they bundle related skills. These are the ones that earn their place after a month of regular use.

🧠 Focus & Structure

feature-dev

Guided feature development that anchors you in codebase context before generating code. Resists premature implementation.

code-review

Specialized agents for PR review. Catches the things you'd miss at 4pm on a Friday.

claude-md-management

Audits and improves CLAUDE.md files. Keeps the project memory file from drifting into stale instructions.

🎨 Creative

frontend-design

Generates distinctive, production-grade frontend interfaces. Avoids the generic AI aesthetic that screams "vibe-coded."

figma

Read and update Figma files from a conversation. Cuts the back-and-forth between design tool and code editor.

adobe-for-creativity

Adobe Creative Cloud bridge — useful for asset workflows that still live in Adobe land.

📚 Research

context7

Fetches up-to-date library documentation on demand. Stops Claude from inventing API signatures from old training data.

exa

Bundles the Exa MCP plus research-tuned skills. The full research stack in one install.

explanatory-output-style

Forces Claude to explain its reasoning at each step. Excellent for teaching sessions or when you need to learn alongside the output.

⚙ Automation

agent-sdk-dev

Scaffolds a new Agent SDK application end-to-end. The fastest way into standalone agent work.

playwright

Bundles the Playwright MCP plus prompts for common automation tasks — smoke tests, scraping, visual diff.

hookify

Build pre-tool, post-tool, and stop hooks from natural language. Easier than editing settings.json by hand.

💬 Communication

slack

Slack MCP plus four slash commands: summarize-channel, find-discussions, standup, channel-digest.

circleback

Meeting intelligence into your conversation. Turns "what did we decide?" into a one-line answer.

🚀 Deployment

firebase

Spin up a Firestore-backed app, manage hosting, deploy Functions — without leaving Claude Code.

Gaps worth filling

🔍
If you don't have these yet, consider them
  • GitHub MCP — most code workflows leak time on tab-switching to the GitHub UI. github@claude-plugins-official closes that loop.
  • Exa — if you do research-heavy work and currently rely on raw WebFetch, Exa pays for itself in cleaner results within a week.
  • Figma — for anyone whose work crosses into design files, the round-trip cost without a Figma plugin is real.
  • explanatory-output-style — if it's installed but disabled in settings.json, re-enabling it for teaching or self-learning sessions is a free upgrade.
