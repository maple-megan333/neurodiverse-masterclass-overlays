Step 4: Turn on Memory

🧠
Why this matters for ND brains
Re-explaining your context every conversation is a cognitive tax. Memory eliminates that.
🟣
Claude
Settings → Memory → ON
Say "Remember that I prefer direct communication"
🟢
ChatGPT
Profile → Personalization → Memory → ON
Say "What do you remember?" to check
🔵
Gemini
No persistent memory yet. Use Custom Instructions instead.

Step 5: Create Your First Project

✏️
Do this now (Claude)
  1. Click Projects in sidebar
  2. Click New Project
  3. Name it: "My AI Setup"
  4. Add instructions: "This project tracks my AI learning journey."

Step 6: Write Custom Instructions

💡
Better with onboarding data
For best results, complete the Onboarding Interview first, then return here to generate custom instructions with your real data.
✏️
Do this now
Based on what you know about me, draft custom instructions for my AI settings. Include:
1. Communication style (how I like to receive information)
2. My context (role, priorities, what I use AI for)
3. ND accommodations (shorter paragraphs, no shame language, low-energy options)
4. Output preferences (format, length, tone)

Keep it under 500 words. Make it sound like ME.
