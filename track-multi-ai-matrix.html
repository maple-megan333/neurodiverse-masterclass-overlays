How to use this matrix

🔎
Read the matrix as a starting point, not a verdict

Pricing and benchmarks shift quickly; verify before subscribing to a paid tier. One option: bookmark this page and revisit each quarter.

When tool data is suspicious

🛡️
Run a second-opinion pass before trusting a critical number

Every tool in this matrix hallucinates sometimes. The Perplexity citation may point at a real URL that says something different. The Claude long-PDF summary may miss a clause buried in section 14. The ChatGPT data analysis may quietly drop a row.

When a number, citation, or claim is going to drive a real decision — a contract, a financial move, a healthcare choice — run it through a second model in the matrix and check the source directly. If the answer crosses a money, medical, or legal threshold, also point the security-reviewer skill at the output for a structured red-team pass.
