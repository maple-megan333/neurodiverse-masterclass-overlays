What Does "Connecting" Mean?

Right now, your AI lives in a browser tab. It can't see your files, your calendar, your Notion databases, or your email. Every time you want AI to help with something, you have to copy-paste context into the chat and copy-paste results back out.

Connecting changes that. When your AI is connected to your tools, it can:

  • Read your Notion databases and add entries directly
  • Access files on your computer without you uploading them
  • Search the web for current information
  • Read and draft emails
  • Check your calendar and schedule events

There are multiple ways to do this, from zero-setup to fully automated. Pick the level that matches your energy and technical comfort today — you can always level up later.

📋 Option 1: Manual Copy-Paste (Zero Setup)

No setup. No accounts. No configuration. You just use AI in one tab and your tools in another.

How it works

  1. Open your AI tool (Claude, ChatGPT, or Gemini) in one browser tab
  2. Open Notion (or whatever you use) in another tab
  3. Copy relevant context from Notion → paste into AI chat
  4. Get AI's response → copy the parts you want → paste back into Notion

When to use this

  • You're just getting started and want to try AI before committing to setup
  • Low energy day — you don't want to troubleshoot configuration
  • You only use AI occasionally
  • You prefer having full control over what goes where

Tips to make it smoother

  • Keep both tabs pinned so they're always one click away
  • Use keyboard shortcuts: Ctrl+A (select all), Ctrl+C (copy), Ctrl+V (paste)
  • When pasting into AI, start with "Here's the context:" so the AI knows what it's looking at
  • Ask AI to format its output for easy pasting — "Format this as a Notion-friendly bulleted list"

Best for: Everyone starting out. There's zero shame in this approach — it works.

🟣 Option 2: Claude Projects — Give Claude Persistent Memory (5 min)

Claude Projects let you create a workspace where Claude remembers your context across conversations. Think of it like giving Claude a folder with everything it needs to know about you.

What you need

  • A Claude account (free or Pro at claude.ai)

Step-by-step setup

  1. Go to claude.ai and sign in
  2. In the left sidebar, click Projects
  3. Click Create a Project
  4. Name it "Neurodiverse AI MasterClass"
  5. Click on your new project to open it
  6. Click Project Instructions (the gear icon) and paste this: 
    You are my AI assistant configured for my neurodivergent brain.

About me:
- [Paste your AI Profile here — or fill this in later]
- I prefer structured responses with clear headers
- Break long tasks into small, numbered steps
- Check in before making assumptions about what I want
- If I seem overwhelmed, offer a simpler version

When I ask you to save something, format it so I can paste it directly into Notion.
When I share a prompt that works, help me refine and save it to my prompt library.
  7. You can also upload files to the project — your AI Profile doc, reference materials, anything Claude should have access to

How to use it daily

  • Always start new conversations inside the project (not from the main Claude screen)
  • Claude will automatically have your instructions and uploaded files as context
  • You can create multiple projects — one for work, one for personal, one for this MasterClass

Pro tips

  • Upload your AI Profile as a file so Claude can reference it
  • Add a "Voice Card" file so Claude writes in your style (see AI Writing Voice)
  • Add frequently-used prompts as files so you can say "use the email template" instead of re-pasting
✅
You're done when
You start a new chat in your project and Claude already knows who you are, how you communicate, and what you're working on — without you explaining anything.
🟢 Option 3: ChatGPT — Custom GPTs and Memory (10 min)

ChatGPT has two ways to personalize: Memory (free) and Custom GPTs (Plus). Both help ChatGPT remember how you work.

Part A: Turn on Memory (free, 2 minutes)

  1. Go to chat.openai.com and sign in
  2. Click your profile icon (bottom-left) → Settings
  3. Go to Personalization → Toggle Memory to ON
  4. Go to Custom Instructions and fill in:
    • "What would you like ChatGPT to know about you?"
      Paste: "I'm neurodivergent (ADHD/autism). I need structured responses with headers. Break tasks into small steps. Don't assume — ask. If I seem stuck, offer a low-energy version."
    • "How would you like ChatGPT to respond?"
      Paste: "Use bullet points and numbered lists. Lead with the answer, then explain. Keep paragraphs short. Bold important terms. If something has multiple steps, number them."
  5. Click Save

Part B: Create a Custom GPT (Plus only, 10 minutes)

  1. Go to chat.openai.com/gpts/editor
  2. Click Create
  3. Switch to the Configure tab (more control than the chat-based builder)
  4. Fill in:
    • Name: "My ND Assistant"
    • Description: "Personal assistant configured for neurodivergent workflows"
    • Instructions: Paste your AI Profile and the custom instructions from Part A
    • Conversation starters: Add 3-4 prompts you use often, like:
      "Help me break down this task into steps"
      "Draft an email based on these bullet points"
      "What should I focus on today?"
  5. Under Capabilities, toggle ON:
    • Web Browsing (search the internet)
    • DALL-E Image Generation (if you want it)
    • Code Interpreter (runs Python, analyzes files)
  6. Under Knowledge, upload files — your AI Profile, voice card, reference docs
  7. Click Create → choose "Only me"

How to use daily

  • Find your Custom GPT in the sidebar under "My GPTs"
  • Click it to start a new conversation — it already has your context
  • ChatGPT's memory builds over time. After a few conversations, it'll remember your preferences even in regular chats
✅
You're done when
Your Custom GPT greets you by context (not by name) and responds in your preferred format without being told.
🔵 Option 4: Google Gemini + Workspace (5 min)

If you live in Google's ecosystem (Gmail, Docs, Drive, Calendar), Gemini can connect directly to all of it. No API keys, no configuration files — just toggle switches.

Step-by-step setup

  1. Go to gemini.google.com and sign in with your Google account
  2. Click the Settings gear icon (top-right)
  3. Click Extensions
  4. Enable these:
    • Google Workspace — Gemini can read your Gmail, Docs, Sheets, Slides, and Drive files
    • Google Maps — ask about locations, directions, nearby places
    • YouTube — summarize videos, find tutorials
    • Google Flights — search flights (handy for travel)
    • Google Hotels — search hotels
  5. Go back to Settings → Gems (Gemini's version of custom assistants)
  6. Click Create a Gem
  7. Give it instructions similar to the Claude Projects setup: 
    You are my ND-friendly assistant. I'm neurodivergent.
- Keep responses structured with headers
- Break tasks into small steps
- When I reference my email or docs, search my Workspace
- Format outputs so I can paste them into Notion
  8. Save your Gem

Things you can now do

  • "@Gmail summarize my unread emails from today" — Gemini reads your inbox
  • "@Docs find my document about project planning" — searches your Drive
  • "@YouTube find a tutorial on MCP setup" — searches and summarizes videos
  • "Look at my last 5 emails from [person] and draft a reply"
  • "What's on my calendar tomorrow and what should I prepare?"

Limitations

  • Gemini can read Google Workspace but can't write to it yet (can't send emails or edit docs directly)
  • It can't connect to Notion — you'll still need to copy-paste between Gemini and Notion
  • Extensions are only available in English in supported countries
✅
You're done when
You can ask Gemini "What emails did I get today?" and it actually shows you your real emails.
🔮 Option 5: Claude Desktop + MCP — Full Power Mode (15-20 min)

This is the most powerful setup. MCP (Model Context Protocol) is an open standard created by Anthropic that lets Claude connect to almost anything — Notion, your files, Google Calendar, Slack, databases, and more. Claude can read AND write, meaning it can create Notion pages, save files, and take real actions.

What you need before starting

  • A computer (Mac, Windows, or Linux)
  • Claude Pro or Team subscription ($20/month at claude.ai)
  • Node.js installed (needed to run MCP servers) — download from nodejs.org (choose the LTS version)
  • Your Notion account (free works fine)

Step 1: Install Claude Desktop (2 min)

  1. Go to claude.ai/download
  2. Download the app for your operating system
  3. Install it and sign in with your Claude account
  4. Verify it works — open the app and send a test message

Step 2: Install Node.js if you don't have it (3 min)

  1. Open your terminal:
    • Mac: Open Spotlight (Cmd+Space), type "Terminal", hit Enter
    • Windows: Press Win+R, type "cmd", hit Enter
  2. Check if Node.js is installed: type node --version and hit Enter
  3. If you see a version number (like v20.11.0), you're good — skip to Step 3
  4. If you get an error, go to nodejs.org, download the LTS version, and install it
  5. Close and reopen your terminal, then verify: node --version

Step 3: Create your Notion integration (3 min)

  1. Go to notion.so/profile/integrations
  2. Click New Integration
  3. Fill in:
    • Name: "Claude MCP"
    • Associated workspace: Select your MasterClass workspace
    • Capabilities: Check all boxes (Read, Update, Insert content)
  4. Click Save
  5. You'll see an Internal Integration Secret — it starts with ntn_. Copy this and save it somewhere safe (you'll need it in Step 4)
  6. Important: Now go to each Notion page or database you want Claude to access:
    • Click the ··· menu (top-right of the page)
    • Click Connections
    • Search for "Claude MCP" and add it
    • Repeat for every page/database Claude should see

Step 4: Configure MCP in Claude Desktop (5 min)

  1. Open Claude Desktop
  2. Go to Claude menu → Settings
  3. Click Developer in the left sidebar
  4. Click Edit Config — this opens a JSON config file
  5. Replace the entire contents with this (then customize the values marked with CAPS): 
    {
  "mcpServers": {
    "notion": {
      "command": "npx",
      "args": ["-y", "@notionhq/notion-mcp-server"],
      "env": {
        "OPENAPI_MCP_HEADERS": "{\"Authorization\": \"Bearer YOUR_NOTION_KEY_HERE\", \"Notion-Version\": \"2022-06-28\"}"
      }
    },
    "filesystem": {
      "command": "npx",
      "args": [
        "-y",
        "@modelcontextprotocol/server-filesystem",
        "YOUR_FOLDER_PATH_HERE"
      ]
    }
  }
}
  6. Replace YOUR_NOTION_KEY_HERE with the key from Step 3 (the one starting with ntn_)
  7. Replace YOUR_FOLDER_PATH_HERE with a folder path:
    • Mac: /Users/yourname/Documents
    • Windows: C:\\Users\\yourname\\Documents (use double backslashes)
  8. Save the file
  9. Quit Claude Desktop completely (not just close the window — right-click the dock/taskbar icon and Quit)
  10. Reopen Claude Desktop

Step 5: Verify it's working (2 min)

  1. Start a new conversation in Claude Desktop
  2. Look for a 🔌 plug icon or MCP tools indicator near the text input — this means servers are connected
  3. Try these test messages: 
    Search my Notion for any page with "AI" in the title
    List the files in my Documents folder
    Create a new Notion page called "Test Page - Delete Me" with the content "MCP is working!"
  4. If the search works and the page appears in Notion — you're connected!
  5. Delete the test page when you're done

Want more connections? Add these MCP servers

You can add more servers to the config file. Here are popular ones — just add them inside the "mcpServers" object, separated by commas:

Google Calendar:

"google-calendar": {
  "command": "npx",
  "args": ["-y", "@anthropic-ai/mcp-server-google-calendar"]
}

Web Search (Brave):

"brave-search": {
  "command": "npx",
  "args": ["-y", "@anthropic-ai/mcp-server-brave-search"],
  "env": {
    "BRAVE_API_KEY": "YOUR_BRAVE_KEY"
  }
}

(Get a free Brave Search API key at brave.com/search/api)

GitHub:

"github": {
  "command": "npx",
  "args": ["-y", "@modelcontextprotocol/server-github"],
  "env": {
    "GITHUB_PERSONAL_ACCESS_TOKEN": "YOUR_GITHUB_TOKEN"
  }
}

Troubleshooting

ProblemFix
No plug icon shows upMake sure you fully quit and reopened Claude Desktop (not just closed the window). Check the config file for JSON syntax errors — missing commas or brackets break everything.
"Command not found: npx"Node.js isn't installed or isn't in your PATH. Reinstall from nodejs.org and restart your terminal.
Notion says "not authorized"You need to share each Notion page/database with your integration. Go to the page → ··· → Connections → add "Claude MCP".
Config file won't saveMake sure the JSON is valid. Common mistakes: trailing commas, missing quotes, wrong slash direction on Windows (use \\ not /).
Server keeps disconnectingThis can happen if npx is downloading packages each time. Try running npx -y @notionhq/notion-mcp-server in your terminal first to cache the package.

💻 Option 6: Claude Code — AI in Your Terminal (15 min, technical)

Claude Code is a command-line tool for people comfortable with a terminal. It gives Claude full access to your project files, can run code, and supports all MCP servers. It's the most powerful option but requires some technical comfort.

Who this is for

  • You use a terminal/command line already
  • You work with code, scripts, or text files
  • You want the maximum AI capability available

Step 1: Install (2 min)

Open your terminal and run:

npm install -g @anthropic-ai/claude-code

If you get a permissions error on Mac, prefix with sudo:

sudo npm install -g @anthropic-ai/claude-code

Step 2: Authenticate (1 min)

claude login

This opens your browser. Sign in with your Anthropic account. Once authenticated, the terminal will confirm you're logged in.

Step 3: Start using it (immediate)

# Go to any project folder
cd ~/Documents/my-project

# Launch Claude Code
claude

# You're now in an AI-powered terminal. Try:
> what files are in this project?
> read the README and summarize it
> create a new file called ideas.md with 5 project ideas
> find every file that mentions "TODO"

Step 4: Add MCP servers for external tools

Claude Code can connect to the same MCP servers as Claude Desktop:

# Connect to Notion
claude mcp add notion -- npx -y @notionhq/notion-mcp-server \
  --env OPENAPI_MCP_HEADERS='{"Authorization":"Bearer YOUR_NOTION_KEY","Notion-Version":"2022-06-28"}'

# Connect to web search
claude mcp add brave-search -- npx -y @anthropic-ai/mcp-server-brave-search \
  --env BRAVE_API_KEY=YOUR_BRAVE_KEY

# See what's connected
claude mcp list

Useful commands

CommandWhat it does
claudeStart interactive mode in current folder
claude "do something"Run a one-shot command
claude mcp listShow connected MCP servers
claude mcp add [name]Add a new MCP server
/helpShow help inside Claude Code
/clearClear conversation history
Ctrl+CCancel current operation
/exitQuit Claude Code

⚡ Option 7: Cowork Mode — AI Desktop Assistant (newest)

Cowork is a feature in the Claude desktop app that gives Claude access to your computer's file system, a code sandbox, web browsing, and connected apps — all through a chat interface. No terminal required.

What you need

  • Claude desktop app (download from claude.ai/download)
  • Claude Pro or Team subscription

How to start

  1. Open the Claude desktop app
  2. Click Cowork in the sidebar (or start a new Cowork session)
  3. Select a folder on your computer that Claude can access
  4. Claude can now read/write files in that folder, run code, browse the web, and connect to apps

Connecting apps in Cowork

Cowork supports MCP connectors for popular apps. To connect them:

  1. In a Cowork session, ask Claude: "What connectors are available?"
  2. Claude will search and suggest relevant connectors (Notion, Google Calendar, Slack, etc.)
  3. Follow the prompts to install and authenticate each connector

What you can do

  • "Read the files in my selected folder and summarize what's there"
  • "Create a spreadsheet comparing these three options"
  • "Search my Notion for meeting notes from last week" (with Notion connected)
  • "Check my calendar for tomorrow" (with Google Calendar connected)
  • "Write a Python script that processes this CSV file and run it"
✅
You're done when
You can chat with Claude in Cowork and it accesses your files, creates documents, and connects to your apps.

Voice Mode

Claude supports Voice Mode in the desktop and mobile apps. Tap the microphone to talk instead of typing — great for brainstorming or when typing feels like too much.

  • Works in regular chat and Cowork sessions
  • Useful for ND users who think better out loud
  • Claude responds with text (or voice on mobile)
