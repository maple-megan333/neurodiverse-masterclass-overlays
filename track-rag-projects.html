How Layer 1 works

You drop files into a folder inside a chat tool. The tool indexes them in the background, then routes every question through that index before answering. No code, no embeddings to manage, no vector database to host. You get RAG, you don't think about RAG.

The five tools below all do this. The difference is what kind of file mix and what kind of question each one handles best — not which one is "better."

The 5 tools, in plain comparison

Claude Projects Default pick
File limit: 30MB / file
Context: 200K tokens (~500 pages)
Audio: No
Plan: Pro $20/mo

The general-purpose RAG choice. You upload files, write custom instructions ("always reply in 3-bullet summaries, never use the word 'leverage'"), and every new conversation in that Project inherits both the files and the instructions.

The hidden superpower: at scale Claude auto-activates RAG mode, expanding effective capacity by roughly 10×. Unlike most retrieval tools, Claude reasons across documents instead of only pulling matching chunks — useful when the answer requires connecting two different sources.

Known quirk: The RAG-mode trigger fires on file count, not token size (Issue #25759), so a Project with 100 tiny files behaves differently from a Project with 5 big ones. If retrieval feels off, try consolidating into fewer, longer files.

Custom instructions = define your format preferences once. Every session in this Project then arrives pre-configured. No more retyping "explain like I'm tired" at the start of each chat.
NotebookLM Audio Overview ★
Sources: 50 per notebook
Per source: 500K words
Context: 1M tokens
Plan: Free tier exists

Google's source-grounded research tool. The strictest RAG of the bunch — NotebookLM refuses to answer outside your uploaded material. Ask it about something you didn't give it and it will tell you it can't, instead of inventing.

That refusal makes it the closest thing to zero-hallucination RAG available in a consumer product. For study, research synthesis, and "I need to actually understand these 30 PDFs," that strictness is the feature.

Audio Overviews turn any notebook into a podcast-style conversation between two AI hosts discussing your sources. Four formats, 80+ languages, generated in a few minutes. Free.

If reading PDFs feels like wading through molasses, Audio Overviews are the cheat code. Upload your sources, generate an audio file, walk around while it explains the material to you. No other tool on this list does this.
ChatGPT Projects Voice mode
File limit: Varies by plan
Sources: Large (plan-tiered)
Audio: Voice input
Plan: Plus $20/mo

Projects, not Custom GPTs — these are two different OpenAI features and the distinction matters. Projects are private workspaces that hold files plus per-project instructions for your own use. Custom GPTs are shareable bots in the GPT Store. If you're doing personal RAG, you want Projects.

The standout is voice integration: open the iPhone or Android app, tap the mic, and you can have a spoken conversation that's grounded in your uploaded files. Speech-to-text input combined with retrieval makes this the tool of choice when typing is fatiguing or when you're processing thoughts out loud.

Voice mode pairs naturally with body-doubling. Put your phone on the desk, talk through a document you uploaded, let ChatGPT play back what you said and summarize. The barrier to "starting" drops dramatically.
Gemini Gems Live Drive sync
File limit: 10 files / Gem
Sources: 10 (live Drive references)
Audio: No
Plan: Google Workspace

Gemini's "Gem" is a saved assistant configuration with instructions and files attached. The file limit is small (10), but the unique trick is that those 10 slots can point to live Google Drive documents instead of static uploads.

Translation: if you edit the Drive doc, the Gem references the new version on the next question. No re-uploading after every revision. For people whose source-of-truth notes already live in Docs and Sheets, this single feature can outweigh smaller limits.

If you already work in Google Drive, Gemini Gems is the lowest-friction option. Zero "I need to remember to re-upload after I edit." Lower memory tax.
Perplexity Spaces Web + private docs
File limit: 50 files
Sources: 50 + live web
Audio: No
Plan: Pro $20/mo

The only Layer 1 tool that natively combines your private documents with live web search in the same query. Ask a question and Perplexity will pull from your uploaded files and fresh sources from the open web, citing both.

Particularly good for small teams doing research where "what's already in our internal docs?" and "what's the current state of the field?" need to be answered together. Spaces can be shared with collaborators, which makes it the closest thing to multi-user RAG without graduating to Layer 3.

Built-in citations on every answer reduce the verify-claims tax. You spend less effort second-guessing whether something was hallucinated.

Citation reliability test

🔍
The 10-question spot-check

Before you trust any RAG setup with real work, run a test you can verify. Pick 10 questions you already know the answer to from the documents you uploaded. Ask them one at a time. For each answer, check:

  1. Is the cited source the correct document?
  2. Does the quoted passage actually say what the AI claims it says?
  3. Is the answer factually right?

Score the results:

  • 9–10 / 10 correct — reliable. Use it.
  • 7–8 / 10 — usable for low-stakes drafting, verify everything for high-stakes work.
  • Below 7 / 10 — broken. Try fewer files, consolidate, or switch tools.

This single 20-minute test will save more time than any amount of reading reviews. It's the production-grade evaluation method (RAGAS) compressed into a non-developer-friendly proxy.

Privacy when using Layer 1 cloud tools

All five tools above run in the cloud. Default settings vary — some opt you into model training, some don't. The opt-out steps as of mid-2026:

🔒
Per-tool opt-out
  • Claude — Free/Pro/Max do not train on your conversations by default. No action needed for personal accounts. Claude Team tier adds contractual guarantees if you handle client work or sensitive material.
  • NotebookLM — Personal Google accounts: your uploads are not used for training. Workspace accounts inherit your admin's data-processing terms.
  • ChatGPT — Settings → Data Controls → toggle "Improve the model for everyone" OFF. Or upgrade to ChatGPT Team / Enterprise, which excludes training by default.
  • Gemini — myactivity.google.com → Gemini Apps Activity → pause. Workspace accounts: admin-controlled.
  • Perplexity — Settings → Account → AI Data Retention → toggle off. Pro accounts get stricter retention.

For anything that wouldn't be okay public — even with the toggles off — use a Layer 2 local tool instead. Cloud toggles are policy promises, not technical guarantees.
